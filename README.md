# Veryable Coding Challenge
The objective of this coding challenge is to build an app utilizing Json to showcase a list of
banking and card accounts. 

# Project Details 
The project is setup using MVVM software design pattern:
- Account (Model), View (Activities), ViewModel(Activity View Models)

Below are some more aspects used:
- Retrofit (Gson)
- Adapter
- RecyclerView
- DataBinding
- Scalable SDP Dimensions

Things to improve upon:
- Loading the bank and card lists quicker
- Including coroutines 
- Using onClickListener interface
